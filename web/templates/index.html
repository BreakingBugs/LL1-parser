Especificaciones de la Gramática

Las producciones siguen el siguiente formato:

Start -> A
A -> ( A ) | Two
Two -> a
Two -> b

Se utiliza "->" para separar el no-terminal y el cuerpo de la produccion.

x -> y | z

es el atajo EBNF de:

x -> y
x -> z

Importante: Cada cadena sin espacios se considera un símbolo gramatical. Ej.: id ≠ i d
